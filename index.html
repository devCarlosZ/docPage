Introduction

At its heart, HTML is a fairly simple language made up of elements, which can be applied to pieces of text to give them different meaning in a document.

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. With Cascading Style Sheets (CSS) and JavaScript, it forms a triad of cornerstone technologies for the World Wide Web.

HTML Documents

HTML (HyperText Markup Language) is the most basic building block of the Web. It defines the meaning and structure of web content. Other technologies besides HTML are generally used to describe a web page's appearance/presentation (CSS) or functionality/behavior (JavaScript).

  • All HTML documents must start with a document type declaration: <!DOCTYPE html>.
  • The HTML document itself begins with <html> and ends with </html>.
  • The visible part of the HTML document is between <body> and </body>.
    • <!DOCTYPE html> <html> <body> <h1>My First Heading</h1> <p>My first paragraph.</p> </body> </html>
    HTML Headings

    HTML headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading. <h6> defines the least important heading.

  • Heading information may be used by user agents, for example, to construct a table of contents for a document automatically.
  • Do not use lower levels to decrease heading font size: use the CSS font-size property instead.
  • Avoid skipping heading levels: always start from <h1>, next use <2> and so on.
  • You should consider avoiding using <h1> more than once on a page. See Defining sections in Using HTML sections and outlines for more information.
    • <h1>This is heading 1</h1> <h2>This is heading 2</h2> <h3>This is heading 3</h3>
    HTML Paragraphs

    HTML paragraphs are defined with the <p> tag.

    <p>This is a paragraph.</p> <p>This is another paragraph.</p>
    HTML Links

    HTML links are defined with the <a> tag.

    <a href="https://www.w3schools.com">This is a link</a>
    HTML Images

    HTML images are defined with the <img> tag. The source file (src), alternative text (alt), width, and height are provideed as atrributes.

    <img src="w3schools.jpg" alt="W3Schools.com" width="104" height="142">
    HTML Buttons

    HTML buttons are defined with the <button> tag.

    <button>Click me</button>
    HTML List

    HTML list are defined with the <ul> (unordered/bullet list) or the <li> (ordered/numbered list) tag, followed by the <li> tags (list items).

    <ul> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </ul>
